1. What does the script do?
	Count the size of the .text/.data/.bss ... segment in the .map file. 
	
2. Usage:
	python map_parser.py <map file>
	
3. NOTE:
	1) Only support to .map files generated by arm-gcc.
	2) Supported linker files:
		- MIMXRT1176xxxxx_cm7_flexspi_nor.ld
		- MIMXRT1062xxxxx_flexspi_nor.ld
		- RW610_flash.ld
